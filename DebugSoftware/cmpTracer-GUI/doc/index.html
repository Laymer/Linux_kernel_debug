Control buttons
  • Show RSS : Displays the Maximum Memory Resident Set Size
  • Show Execution time : Displays overhead associated with each tracer
  • File size : Tracers generate different traces with different sizes
  • Show Voluntary Context Switches : Shows the number of voluntary context switching
  • Show Involuntary Context Switches : Shows the number of involuntary context switching
  • Minor page faults : Minor page faults overhead produced by different tracers.
