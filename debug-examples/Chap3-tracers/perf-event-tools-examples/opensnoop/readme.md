# Chap3-tracers/perf-event-tools-examples/killsnoop

Example to illustrate the use of `opensnoop` to trace `open` system calls.

# Usage

Compile program's sources :
> gcc openfile.c -o openfile

Launch **opensnoop** :
> sudo ./opensnoop

Start the program :
> gcc ./openfile
 
Take a look at **opensnoop** output.
